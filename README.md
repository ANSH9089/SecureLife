# SecureLife

## Overview

This Android application is designed to streamline the management of residents, security guards, and administrators in a residential community. It includes features for user registration, role assignment, visitor management, notice boards, and a separate chat application for residents.

## Features

1. **User Registration and Role Assignment:**
   - New users can register in the app.
   - Admin can assign roles (Admin, Security Guard, Resident) to registered users.
   - If a user is assigned the role of a resident, the admin can also assign a block number.

2. **Admin Privileges:**
   - Remove members from the system.
   - Add notices for residents.

3. **Security Guard Functionality:**
   - Fill out a form for new visitors.
   - Residents receive visitor details and can either accept or ignore the request.
   - Security guards receive notifications to manage visitor access.

4. **Resident Features:**
   - Separate chat application for residents.
   - View the location of visitors on the map.
   - Manage visitor access by allowing or denying them.
